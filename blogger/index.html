Disable date/time sync between VirtualBox client and host

by msa.operation @ forum.virtualbox.org original post: https://forums.virtualbox.org/viewtopic.php?t=8535 vboxmanage setextradata [VMname] "VBoxInternal/Devices/VMMDev/0/Config/GetHostTimeDisabled" "1" Note: replace [VMname] with the name of target virtual machine; replace “1” with “0” to turn the sync back on. »
